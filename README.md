# Redesain Website Pondok Pesantren Darussalam Tasikmalaya

Proyek ini adalah hasil redesain sistem informasi berbasis web untuk Pondok Pesantren Darussalam Rajapolah Tasikmalaya. Fokus utama adalah modernisasi antarmuka (UI) dan peningkatan pengalaman pengguna (UX).

## Fitur Utama
- **Responsive Design**: Optimal untuk perangkat mobile dan desktop.
- **Modern UI**: Menggunakan palet warna hijau gelap dan kuning emas sesuai logo institusi.
- **Fast Loading**: Optimasi aset dan penggunaan font standar Google.

## Struktur File
- `index.html`: Struktur utama halaman web.
- `style.css`: Kumpulan aturan styling dan desain sistem.
- `script.js`: Logika interaktivitas (seperti menu navigasi mobile).

## Cara Menjalankan
1. Clone repository ini.
2. Buka file `index.html` menggunakan browser pilihan Anda.
